FAQs

  • What is Frontend Mentor, and how will it help me?

    Frontend Mentor offers realistic coding challenges to help developers improve thier frontend coding skills with projects in HTML, CSS and Javascript. It's suitable for all levels and ideal for portfolio building.

  • Is Frontend Mentor free?

    Yes, Frontend Mentor offers both free and premium coding challenges, with the free option providing a range of projects suitable for all skill levels.

  • Can I use Frontend Mentor projects in my portfolio?

    Yes, you can use projects completed on Frontend Mentor in your portfolio. It's an excellent way to showcase your skils to potential employers!

  • How can I get help if I'm stuck on a Frontend Mentor challege?

    The best place to get help is inside Frontend Mentor's discord community. There's a help channel where you can ask questions and seek support from other community members.
